# 🏥 豪斯醫生：診斷王國 (House M.D. Mobile)

這是一個專為手機設計的《豪斯醫生》模擬 App。結合了迪士尼城堡背景與 PPTH 診斷室的黑暗幽默。

## 🚀 專案特點
- **行動優化**：專為手機螢幕設計，支援「加入主畫面」成為 Web App。
- **載入動畫**：啟動時會有專屬的豪斯王國圖標與診斷中動畫。
- **互動機制**：
  - 處理 20 位童話病患的疑難雜症。
  - 管理豪斯的疼痛值（維柯丁系統）。
  - 應付隨機出現的門診任務（Cuddy 的逆襲）。

## 📱 如何安裝到手機？
1. 用手機瀏覽器打開 GitHub Pages 網址。
2. 點擊瀏覽器的「分享」或「選單」按鈕。
3. 選擇 **「加入主畫面」** (Add to Home Screen)。
4. 像一般 App 一樣從桌面啟動。
